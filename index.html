VortexAI v6.0

          
Designed & built by Zelvior · Pakistan · Apache 2.0

        

      

      
A fully free, production-grade AI chat interface — single HTML file, zero backend, Apache 2.0 license.

    

    

      
v6.0 New Features

      

        
▸Command Palette (Ctrl+P) — Instant access to actions, chats, models.

        
▸Text-to-Speech — Per-message read-aloud + auto-TTS mode.

        
▸Model Favorites — Star models, filter by favorites, pinned badges.

        
▸Diff Tab — Canvas code diff view with snapshot comparison.

        
▸Light Mode — Full light theme for all components.

        
▸History Search — Instant search across all saved chats.

        
▸Import/Export History — Backup and restore all chats as JSON.

        
▸3 New Export Formats — HTML, TXT, plus existing MD and JSON.

        
▸Configurable Max Tokens — Set response length in settings.

        
▸Model Sorting — Sort by name, context length, or recently used.

        
▸Large/Small Filters — Filter models by parameter count.

        
▸Context Ring — Visual token usage on model button.

        
▸Character Count — Live input character counter with color warnings.

        
▸Canvas Format Button — Auto-format code in editor.

        
▸Open Preview in Tab — Launch canvas HTML in a real browser tab.

        
▸Message Pinning — Pin important messages.

        
▸Sidebar Stats — Total chats and messages display.

        
▸5 New Themes — Emerald, Sky, Fuchsia + existing palette.

        
▸Custom Username — Personalize the user avatar label.

        
▸Timestamps — Optional per-message time display.
