NexusAI

Designed & built by Zelvior

A fully free, production-grade AI chat interface with live model scanning, code canvas, and mid-chat model switching — all in a single HTML file.

Model Provider

🔗

OpenRouter

All AI models are served through OpenRouter's unified API. NexusAI continuously scans their /api/v1/models endpoint and filters for 100% free-tier models across all modalities.

Features

▸Live Model Scanner — Fetches & categorises all free OpenRouter models on load with type, expertise, and uncensored tags.
▸Mid-Chat Switching — Change the active model at any point; the new model picks up the same conversation context.
▸Smart Code Canvas — Hidden by default; auto-opens with syntax highlighting whenever code is detected in a response.
▸Thinking Indicator — Animated dots + streaming typewriter effect while the model generates.
▸Category Filters — Filter models by Text, Vision, Uncensored, Coding, Reasoning, Audio, Image Gen.
▸Quick-Switch Bar — Recent models shown as pills above the input for instant switching.
▸Temperature Control — Adjustable in Settings.
